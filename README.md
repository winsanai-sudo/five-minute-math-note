# 5분 수학노트

수업 마지막 5분 동안 학생이 오늘 배운 내용, 핵심 개념, 숙제, 질문, 이해도를 기록하고 강사가 제출 현황과 숙제 완료 여부를 확인하는 React 웹앱입니다.

## 로컬 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## Render 배포

이 저장소의 루트에 있는 `render.yaml`을 사용합니다.

- Root Directory: `five-minute-math-note-complete`
- Build Command: `npm install && npm run build`
- Publish Directory: `dist`
